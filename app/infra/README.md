# Infraestructura mínima

## Docker

El archivo `docker-compose.yml` en la raíz levanta frontend, backend y PostgreSQL con una configuración pensada para desarrollo local y presupuesto ajustado.

## Terraform

La carpeta `terraform/` incluye un ejemplo opcional usando el proveedor Docker para construir y ejecutar solo el backend. Es útil como punto de partida barato antes de adoptar una nube administrada.
